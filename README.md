# My-ROUNDS-mod

## Features

### Keybinds

H - Double jump height
J - Regular jump height

### Misc. Stuff

No Overheat!
No Reloading Required!

# Requirements
ROUNDS - the game ( obviously )
MelonLoader

# How to install
Install MelonLoader
Download this repo
Extract everything inside My-ROUNDS-mod-main into your rounds directory
